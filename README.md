# OneTap iOS

iOS 시스템 문제를 한 번의 탭으로 해결하는 소프트웨어 라인업 공식 웹사이트.

## 페이지 구성

- [index.html](index.html) — 메인 (7배너 슬라이더 + 인기 제품 + 후기)
- [복구솔루션.html](복구솔루션.html) — 시스템 복구 & 데이터 복구
- [암호해제.html](암호해제.html) — 잠금 해제 솔루션
- [데이터전송.html](데이터전송.html) — 백업 / 전송 / 미러링
- [베스트세일.html](베스트세일.html) — 한정 특가 (실시간 카운트다운)

## 제품 라인업

| 카테고리 | 제품 |
|---|---|
| 시스템 복구 | OneTap ReBoot · OneTap Recover · OneTap Cleaner |
| 데이터 전송 | OneTap Transfer · OneTap ChatBack · OneTap Mirror |
| 잠금 해제 | OneTap UnLock · OneTap UnLock Pro · OneTap ScreenTime |
| AI 도구 | OneTap Pretty · OneTap PDF AI · OneTap Translator |

## 기술 스택

- 정적 HTML/CSS/JS (외부 의존성 없음)
- 반응형 디자인 (PC / 태블릿 / 모바일)
- 실 사진: Unsplash CDN
