💳 결제·환불

결제 수단은 어떤 것이 있나요?

국내 신용·체크카드(비자/마스터/JCB/아멕스), 카카오페이, 네이버페이, 토스페이, 무통장 입금(가상계좌)을 지원합니다. 해외 카드도 사용 가능합니다.

환불은 언제까지 가능한가요?

구매일로부터 30일 이내에는 무조건 환불해 드립니다. 자세한 조건은 환불 정책에서 확인하실 수 있습니다.

  • 제품이 정상 작동하지 않는 경우
  • 제품이 광고된 기능을 제공하지 못하는 경우
  • 다운로드/설치가 불가능한 경우
자동 갱신을 끄고 싶어요

로그인 후 내 계정 → 구매 내역 → 자동 갱신 해지에서 직접 끄실 수 있습니다. 갱신 예정일 24시간 전까지 해지하시면 다음 결제가 이루어지지 않습니다.

고객센터(1:1 문의)로 요청하셔도 즉시 처리됩니다.

세금계산서·현금영수증을 받을 수 있나요?

네, 모두 발행됩니다. 결제 화면에서 사업자번호 또는 개인 휴대전화 번호를 입력하시면 자동 발행되며, 누락된 경우 7일 이내 고객센터로 요청해 주시면 발행해 드립니다.

⬇️ 다운로드·설치

어디서 다운로드 받나요?

각 제품 페이지의 "무료 다운로드" 버튼 또는 다운로드 센터에서 받으실 수 있습니다. Windows·macOS 두 가지 버전을 제공합니다.

Mac과 Windows 둘 다 지원하나요?

모든 OneTap iOS 제품은 Windows 10·11과 macOS 12 Monterey 이상(Apple Silicon 네이티브 포함)을 지원합니다. 한 번의 구매로 두 OS 모두에서 사용하실 수 있습니다.

설치 중에 백신/스마트스크린이 차단했어요

OneTap iOS는 코드사이닝 인증서로 서명된 안전한 프로그램입니다. 일부 백신이 시스템 깊은 접근을 의심으로 표시할 수 있으니 다음을 시도해 주세요.

  1. 설치 파일을 우클릭 → 속성 → 하단 "차단 해제" 체크
  2. 백신 예외 폴더에 추가
  3. 그래도 실패 시 고객센터로 백신 로그를 보내주세요
설치 후 실행이 안 돼요

다음 순서로 확인해 주세요.

  1. 관리자 권한으로 실행해 보기
  2. 최신 .NET / Visual C++ 재배포 패키지 설치 (Windows)
  3. iTunes 또는 Apple Mobile Device Support 최신 버전 설치
  4. 위 방법으로 안 되면 고객센터에 OS 버전·오류 화면을 첨부해 문의

⚙️ 기능·호환성

iOS 27 베타 / iPhone 17 시리즈도 지원하나요?

네, 모든 제품이 iOS 27 베타 및 iPhone 17 / 17 Air / 17 Pro / Pro Max까지 공식 지원합니다. 신규 iOS 출시 후 평균 48시간 내 호환 업데이트를 배포합니다.

iPad / iPod / Apple Watch도 지원하나요?

iPad와 iPod은 모든 제품에서 지원됩니다. Apple Watch는 데이터 전송(OneTap Transfer)에서만 지원되며, 시스템 복구·잠금 해제는 현재 지원하지 않습니다.

백업 없이 삭제한 데이터를 정말 복구할 수 있나요?

iOS의 강한 암호화 특성상 완전히 영구 삭제된 데이터는 어떤 프로그램으로도 복구할 수 없습니다. OneTap Recover는 다음을 활용해 복구합니다.

  • iTunes/Finder 로컬 백업에서 추출
  • iCloud 백업에서 추출 (Apple ID 로그인 필요)
  • 기기 내부의 캐시·임시 영역에서 미덮어쓰기 데이터 탐색

3번 항목은 삭제 후 시간이 짧을수록 성공률이 높습니다. 무료 스캔으로 복구 가능 여부를 먼저 확인해 보시는 것을 권장합니다.

잠금 해제 기능은 어떤 모델까지 되나요?

화면 비밀번호 해제는 모든 모델에서 가능하지만, iPhone XS(A12) 이상에서는 데이터가 초기화됩니다(Apple의 보안 정책상 불가피). Apple ID 해제는 iPhone X 이전 모델에서만 데이터 유지하며 해제가 가능합니다.

본인 소유의 기기에서만 사용해 주세요. 도난·분실 기기에 사용 시 법적 책임이 따를 수 있습니다.

🔑 라이센스

한 라이센스로 몇 대까지 사용할 수 있나요?

플랜별로 다릅니다.

  • 1개월·1년 플랜: PC 1대, iOS 기기 5대
  • 평생 플랜: PC 1대, iOS 기기 무제한
  • 비즈니스 플랜: PC 5대, iOS 기기 무제한

PC 등록은 자동이며, 재설치 시 자동으로 다시 인증됩니다.

평생 라이센스에 업데이트도 포함되나요?

네, 평생 무료 업데이트가 포함됩니다. 새로운 iOS 버전, 새 iPhone 모델 호환, 기능 추가 모두 추가 비용 없이 받으실 수 있습니다.

라이센스 코드를 잃어버렸어요

로그인 후 내 계정 → 구매 내역에서 언제든 다시 확인하실 수 있습니다. 결제 시 등록한 이메일로도 자동 재발송 받으실 수 있습니다.

다른 PC로 옮기고 싶어요

기존 PC에서 설정 → 라이센스 해제 후 새 PC에 설치·로그인하시면 자동으로 인증됩니다. 1년에 5회까지 무료로 가능합니다.

🔒 데이터·보안

제 데이터에 OneTap iOS가 접근하나요?

모든 데이터 처리는 사용자 PC에서 로컬로 이루어지며, 데이터가 저희 서버로 전송되지 않습니다. 라이센스 인증·업데이트 확인 시에만 최소한의 정보(이메일, 라이센스 코드)만 통신합니다.

결제 정보는 어떻게 보관되나요?

카드 정보는 저희가 직접 보관하지 않고 PG사(Toss / Stripe / 2Checkout)의 PCI-DSS 인증 시스템에 안전하게 토큰화되어 저장됩니다. 자세한 내용은 개인정보 처리방침을 참고해 주세요.

회사 보안 정책은 어떻게 되나요?
  • SSL/TLS 전구간 암호화
  • SOC 2 Type II 인증 (예정)
  • 정보보호관리체계(ISMS) 준수
  • 주기적 외부 보안 감사

🤝 고객지원

한국어 지원이 되나요?

네, 한국인 전담 CS팀이 운영됩니다. 1:1 문의·이메일·전화 모두 한국어로 응대하며, 평균 응답 시간 3시간 이내입니다.

운영 시간은 어떻게 되나요?

1:1 문의·이메일: 24시간 365일 접수 (영업일 기준 평균 3시간 내 답변)
전화 상담: 평일 09:00 – 18:00 (점심시간 12:00 – 13:00 제외)

원격 지원이 가능한가요?

네, 복잡한 문제는 TeamViewer 원격 지원이 무료 제공됩니다. 1:1 문의에서 "원격 지원 요청"으로 신청해 주세요.
